# Momo

**Momo is an ERC4626 vault that buys strategically buys [Hyperdrive](https://github.com/delvtech/hyperdrive) longs and shorts to amplify the returns of a target yield source.**

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Lint

```shell
$ make lint
```

### Format

```shell
$ make prettier
```
